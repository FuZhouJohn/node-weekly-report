# 安装 Install

```
npm i -g @haodademm/nwr
```

# 使用 Use

### 输出本周所有任务 Output Weekly Report

```
t
```

### 添加新任务 Add task

```
t add <reportName>
```

### 删除所有任务 Clear all tasks

```
t clear
```

### 输出本周或所有任务 Output this week or all tasks

```
t list -a
```
